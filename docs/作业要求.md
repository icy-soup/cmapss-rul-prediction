一、作业背景
在当前智能制造浪潮下，预测性维护通过感知装备状态、预判故障并评估剩
余寿命，正从根本上变革传统维修模式，成为降本增效与安全保障的核心使能技
术。在航空航天、能源动力等高端装备领域，关键设备的非计划停机不仅带来巨
大经济损失，更可能引发严重安全风险。因此，精准的剩余使用寿命（RUL）预
测已成为学术界与工业界的共同焦点。航空发动机作为飞机的核心部件，其退化
过程高度非线性，受运行工况、环境条件等多因素耦合影响，个体差异显著。这
种复杂性要求RUL 预测方法既能捕捉长时序依赖，又能建模多元传感器间的协
同退化模式。
近年来，Transformer 架构因出色的长程依赖建模能力被引入时序预测，但标
准Transformer 在处理多变量长序列时面临两大挑战：一是计算复杂度随序列长
度平方增长，难以高效部署；二是传统注意力机制难以显式建模传感器间的关联。
为此，时序数据Patch 化技术将长序列切分为子序列块作为基本单元，既能大幅
降低计算开销，又能捕捉多尺度时序语义。与此同时，以iTransformer 为代表的
变体通过倒置注意力，将变量维度作为token 进行跨变量交互，更直接地捕获多
传感器间的退化关联。结合多尺度Patch 分支并行、传感器分组并行编码等并行
架构，则可进一步兼顾全局感受野与运算效率。
基于此，本作业旨在融合Patch 化表示、Transformer 变体与并行架构，构建
深度RUL预测模型，在CMAPSS数据集上开展实验验证，为航空装备等高端制
造场景下的视情维修与健康管理提供核心技术支撑。
二、作业研究内容

1. 数据预处理与退化分析
   包括：深入理解CMAPSS数据集结构，分析各传感器物理含义及时变特
   性；识别数据中的噪声、工况切换与传感器漂移问题，设计针对性的清洗策
   略；采用滑动窗口法构建有监督样本，确定窗口长度、步长与RUL标签的裁剪
   策略；针对多工况差异，探讨运行条件归一化、自适应标准化等方法；初步分
   析不同传感器对退化的敏感度，为后续特征筛选提供依据。
2. 特征提取与时序Patch 化表示
   包括：从原始多变量时序中提取时域、频域及时频域特征，筛选退化敏感
   特征子集，并探索降维方法；重点研究时序数据的Patch 化处理——将预处理
   后的多通道长序列划分为固定长度的重叠或不重叠Patch，每个Patch 通过线性
   投影或卷积编码为固定维度的Token，作为Transformer 的输入；分析Patch 长
   度、步长、通道独立/共享Embedding 等设计对预测性能的影响；结合
   iTransformer 中“变量作为Token”的思想，探索变量维度Patch 化与时间维度
   patch 化的组合表征方式，以便同时捕获传感器内的时间依赖与传感器间的交互
   关系。
3. 基于Transformer 变体与并行架构的RUL预测模型
   包括：构建以Patch 化时序Token 为输入的核心预测网络，整体采用编码
   器-回归器架构；引入iTransformer 的倒置注意力机制，将传感器变量（或变量
   Patch 组）作为Token 进行跨变量注意力计算，而将时间步作为特征投影，从而
   更直接地建模多传感器退化协同模式；设计并行架构，如多分支时间Patch
   流、传感器分组并行编码等，在保持全局感受野的同时提升计算效率；融合稀
   疏注意力、局部窗口注意力等高效注意力模块，降低长序列建模复杂度；超参
   数优化方面，涵盖Patch 大小、注意力头数、编码器层数、学习率等。
4. 实验验证与结果分析
   包括：在CMAPSS四个子集上开展系统性实验，采用均方根误差
   （RMSE）及官方评分函数进行性能评估；设定基准对比模型，包括传统机器
   学习、经典深度学习、标准Transformer 以及其他先进时序预测模型；完成消融
   实验，分别验证各模块的独立贡献；记录训练收敛曲线、预测值与真实RUL的
   可视化对比；最终总结所提方法的有效性和局限性，提出改进方向。
