# 项目文件目录说明

## 根目录

| 文件 | 用途 |
|------|------|
| `main.py` | 统一实验入口。支持 `--model {base,person1,person2}` 和 `--subset {FD001~FD004}` |
| `vision.js` | 千问 VL 识图脚本，用于审查图片质量问题 |

## docs/ — 文档

| 文件 | 用途 |
|------|------|
| `MSPatch_RUL_技术报告.md` | 详细技术报告（给自己看的），含架构原理、实验结果、消融分析 |
| `CMAPSS_RUL_技术报告.html` | 正式版 HTML 报告，专业格式，不含表情符号 |
| `MSPatch_RUL_Report.md` | （旧版报告，保留） |
| `RUL_iteration_roadmap.docx` | 五人接力迭代架构路线图 |
| `作业要求.md` | 大作业要求说明 |
| `基于深度学习的剩余使用寿命预测.pdf` | 背景文献 |

## src/ — 源代码

### step1_preprocessing/ — 数据加载与预处理

| 文件 | 用途 |
|------|------|
| `loader.py` | 读取 CMAPSS 原始数据，全局/按工况归一化，滑动窗口生成样本 |

### step2_features/ — 特征提取

| 文件 | 用途 |
|------|------|
| `patching.py` | PatchEmbedding（线性投影）和 ConvPatchEmbedding（卷积投影） |

### step3_models/ — 模型定义

| 文件 | 用途 |
|------|------|
| `base_model.py` | Base: Patch-iTransformer-RUL（线性 Patch + iTransformer + MLP） |
| `person1_model.py` | Person 1: ConvPatch-iTransformer-RUL（卷积 Patch + iTransformer + MLP） |
| `person2_model.py` | Person 2: MSPatch-iTransformer-RUL（三分支 ConvPatch + 融合 + iTransformer + MLP） |
| `trainer.py` | 训练器：MSE Loss + AdamW + WarmupCosine + Early Stopping |

### step4_visualization/ — 可视化

| 文件 | 用途 |
|------|------|
| `plot_results.py` | 生成训练曲线、散点图、引擎预测、指标对比等所有图片，支持全模型和精简版两套输出 |
| `plot_ablation.py` | 生成消融实验对比图（分支数量、融合方式、FD004 验证、总结） |

### step5_pipeline/ — 实验流水线

| 文件 | 用途 |
|------|------|
| `ablation.py` | 自动运行 FD001 消融实验（7 种分支配置 + 3 种融合方式），保存结果 |
| `ablation_fd004.py` | 运行 FD004 消融实验（small+medium, single-small, single-medium） |
| `export_excel.py` | 将所有实验结果导出为 xlsx，含 Summary/Ablation FD001/Ablation FD004/Training 四个工作表 |

### system/ — 系统配置与工具

| 文件 | 用途 |
|------|------|
| `config.py` | BaseConfig 数据类：所有超参数默认值 + SUBSET_CONFIG 预设 |
| `metrics.py` | 评估指标实现：RMSE, PHM08 Score, R² |

## results/ — 实验结果

### results/ 根目录

| 文件 | 用途 |
|------|------|
| `summary.csv` | 所有 12 组实验（3 模型 × 4 子集）的 RMSE, R², Score, 参数量汇总 |
| `CMAPSS_Results.xlsx` | Excel 版汇总，含消融实验和训练历史 |

### results/main_experiments/ — 主实验数据

每组实验包含以下 5 个文件（以 `PatchiTransformerRUL_FD001` 为例）：

| 文件类型 | 示例 | 用途 |
|----------|------|------|
| `*_metrics.json` | `PatchiTransformerRUL_FD001_metrics.json` | 评估指标：RMSE, R², Score（JSON 格式） |
| `*_config.json` | `PatchiTransformerRUL_FD001_config.json` | 实验配置参数 |
| `*_history.npz` | `PatchiTransformerRUL_FD001_history.npz` | 训练历史：train_loss, val_loss, val_rmse 等 |
| `*_y_pred.npy` | `PatchiTransformerRUL_FD001_y_pred.npy` | 测试集预测值（NumPy 数组） |
| `*_y_test.npy` | `PatchiTransformerRUL_FD001_y_test.npy` | 测试集真实 RUL 值（NumPy 数组） |

主实验清单：12 组 = 3 模型（Base/P1/P2）× 4 子集（FD001~FD004）

### results/ablation/ — 消融实验数据

| 文件 | 用途 |
|------|------|
| `ablation_summary.csv` | FD001 消融实验汇总（10 组实验的 RMSE, R², Score, 参数量） |
| `ablation_A0-P8S4-FD001_*.json/npy/npz` | 与主实验格式一致的消融结果（A0~A6, B0~B2） |
| `ablation_FD004_*.json/npy/npz` | FD004 消融结果（single-small, single-medium, small+medium） |

### results/figures/ — 图片

| 目录 | 内容 |

| `按子集/全模型/训练曲线/` | FD001~FD004 每子集独立训练曲线（3 模型并排） |
| `按子集/全模型/散点图/` | 每子集独立散点图（3 模型并排） |
| `按子集/全模型/引擎预测/` | 每子集独立引擎预测图（3 模型并排） |
| `按子集/精简(Base+P2)/训练曲线/` | 每子集独立训练曲线（Base+P2，2 模型并排） |
| `按子集/精简(Base+P2)/散点图/` | 每子集独立散点图（Base+P2） |
| `按子集/精简(Base+P2)/引擎预测/` | 每子集独立引擎预测图（Base+P2） |
